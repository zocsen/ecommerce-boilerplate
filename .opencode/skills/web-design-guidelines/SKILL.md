# Web Interface Guidelines

Use this skill for focused UI and UX reviews.

## Workflow

1. Fetch the latest guidelines from `https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md`.
2. Read the target files.
3. Review them against the fetched guidelines.
4. Report issues in terse `file:line` format.

## Use it for

- accessibility reviews
- UI audits
- design quality checks
- UX consistency passes
